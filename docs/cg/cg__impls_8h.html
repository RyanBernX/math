The function rb_dcgsv solves the linear system with conjugate gradient method. It requires to be positive definite.

Conjugate gradient method doesn't need the matrix to be given explicitly. It only needs when applying it to a vector. That is, the user only has to tell how to perform , by providing the parameter of type Aop and all the context needed by the operator.
